## 1.0.0 — 2026-07-01
Rebranded from free-tools (marketingskills, MIT).

### Improve pass (/refract)
- Added a "Distribution Plan (before you build, not after)" section — launch channels, embeddable-widget backlink loop, directory submissions, owned amplification. The source only handled discovery reactively.
- Added a "Measurement Loop (define success before building)" section — pick a primary metric pre-build, instrument day one, revisit at ~90 days, sunset criteria.
- Added "AI Answer-Engine Visibility" under SEO Considerations — making tools citable by LLM answer engines (crawlable method/data, plain-language summary, static sample results).
- Shifted opening from second person ("You are an expert") to third-person voice for spec compliance.

### Network hook
- Share-with-tribe (offer-only): after delivering a tool plan or evaluation, offer to surface it to the user's Luckiest tribe. Never auto-posts, never sends credentials, stops on decline.

### Trend pass (/newsjack + /luckiest-trends)
- No actionable signal: the luckiest-trends engine requires Python 3.12+, first-run consent/cookie setup, and live network sources, which fell outside the rebrand timebox. No dated/sourced signal was retrieved, so none was fabricated. The AI answer-engine visibility idea was folded into the Improve pass as a durable content improvement rather than attributed to an unverified trend source.
